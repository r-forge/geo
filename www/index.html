Links related to the package:

  • Package development on R-Forge
  • The related geoextras package
  • Workshop material from 12 Nov 2010
  • The annual Hafro and NWWG assessment reports contain many geoplots

To install, run: 

install.packages("geo", repos="http://r-forge.r-project.org")

If you catch the R-Forge package server on a bad day, so the normal installation doesn't work, try: 

install.packages("geo", repos="http://r-forge.r-project.org", type="source")
